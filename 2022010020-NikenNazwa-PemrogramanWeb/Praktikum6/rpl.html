Rekayasa Perangkat Lunak (D IV)

Program Studi Rekayasa Perangkat Lunak (RPL) merupakan domain Bidang Ilmu Informatika & Komputer, dalam aspek pengembangan software mulai dari tahap awal spesifikasi, desain, konstruksi, testing sampai pemeliharaan.


Profesi-profesi lulusan RPL antara lain Programmer, System Analyst, Software Quality Assurance Engineer (SQAE), Database Administrator, Software Architect, Software Support, Konsultan IT, Web Designer.


Di indonesia, Program Studi RPL Kurang dari 10. Politeknik Balekambang merupakan satu-satunya Perguruan Tinggi Swasta yang memiliki program studi Rekayasa Perangkat Lunak (Software Enginering). Selain itu, program studi ini mencakup seluruh bidang domain komputer lain.
